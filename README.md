## このソフトウェアでできること
### Windows バッチファイルの記述
最初から
```plane:
@echo off


```
と入力されているので、ひと手間少なくなります。

### バッチの入門(まぢでちょっとだけ)
右に`Windows バッチ・ファイル辞典`というものがあるので、入門したり、忘れたときに調べずに情報を得ることができます。
情報は今後増やしていく予定です。
また、情報は不正確なものである可能性があるため、報告してほしい。
