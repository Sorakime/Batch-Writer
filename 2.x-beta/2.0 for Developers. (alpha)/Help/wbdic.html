Windows バッチ・ファイル辞典

項目 情報
必須項目 知っていると思いますが、Windows Bat(cmd)ファイルを作るときは、必ず最初に
@echo off
と入力しておきましょう。
いちいち実行コマンドがプロンプトに表示されるためです。
文字の出力 文字を出力するときには、bashと同様に
echo
を使用します。そのあとにクォーテーションなしで入力します。これには変数も設定できますが、
%
で変数名を挟む必要があります。
変数の設定 変数を設定するには、Apple Scriptと同様
set 変数名=文字列
とします。クォーテーションは必要ありません。計算した結果も代入できますし、日本語などの文字列も問題なく代入可能です。
ifでの条件分岐 条件分岐をするとき、基本は
if文
を使用します。以下のように書くことができます。
if 条件 (真のときに動かすコマンド) else (偽の時に行うコマンド)
となります。JavaScriptなどとは少し違う書き方をします。
説明が長いので詳細はネットで
gotoでの条件分岐 もう一つの条件分岐やループとして、
goto
を使用します。Go Toキャンペーンではないです。まぁGo Toの意味ですけど。
これはラベル(:から始まるやつ)と組み合わせて使用し、
:label
echo label test
goto :label
というように使用します。		 (編集時に気が付いた、GoTo[ト]ラベル)
forでのループ ループをするには、基本として
for文
を使用します。
for文の書式は、
for /オプション %%アルファベット1つ in (ループの対象) do (
  ループ中に行うコマンド
)

というようになります。結構ややこしいので、開発者は基本goto文のほうを多めに使ってますが、ファイルなどの操作の場合にfor文を重宝します。
引数を扱う ここでは、引数を扱う方法を記述して言います。
引数は、第一引数(起動するためのパス)で
%~0
となり、第二引数以降も同様に0が増えていきます。
これは変数として扱うことができます。
画面の内容を消去する 表示されている画面をクリアするには、
cls
を使用します。一度けしたプロンプトは戻すことは不可能です。
ウィンドウのタイトルを変更する 実行中のウィンドウのタイトル(上にある白い棒の中の文字列)を変えるには、
Title
コマンドを使用します。Titleの後には、変更したい名前を書きます。
ほかのファイルを呼び出す ほかのバッチファイルを呼び出すには、
call
コマンドを使用します。具体的には、
call ドライブ:パス パラメータ
と使用します。ちなみに呼び出されたファイルに
exit
コマンドが書かれている場合、終了した後に呼び出し元のバッチファイルが実行されます。
ディレクトリを表示する 大体のプロンプトである、
dir
コマンド。ディレクトリを表示させます。
これは実際に表示させたほうが速いでしょう。WindowsキーとRを押してcmdと入力し、
dir C:\
と入力してみましょう。Cドライブの内容が表示されます。
